# Culturama

## Sobre o projeto
Este projeto é uma interface de um site de eventos culturais chamado **Culturama**. A aplicação apresenta diferentes tipos de eventos como dança, teatro, festivais, cinema e outras atividades culturais.

O projeto foi desenvolvido com base em um curso realizado na plataforma Alura, com o objetivo de praticar conceitos de desenvolvimento front-end, organização de layout e estruturação de componentes visuais de uma página web.

A proposta do site é centralizar informações sobre eventos culturais, permitindo que o usuário navegue por categorias, visualize destaques e encontre eventos futuros.

### Cabeçalho
O cabeçalho contém:

- Logotipo do site Culturama
- Campo de busca para procurar eventos
- Navegação com opções como:
  - Programação
  - Categorias
  - Seleção de localização do usuário

### Banner principal
A primeira área visual da página apresenta um destaque visual com imagens relacionadas a eventos culturais, como dança, fotografia e teatro. Essa seção funciona como uma vitrine inicial do conteúdo do site.

### Categorias
Logo abaixo do banner principal existe uma seção de categorias que permite navegar pelos tipos de eventos disponíveis:

- Shows
- Arte
- Teatro
- Fotografia
- Festivais
- Tecnologia
- Cinema
- Design

Cada categoria possui uma cor própria para facilitar a identificação visual.

### Destaques
A seção de destaques apresenta eventos que estão em evidência no momento. Normalmente essa área utiliza um banner horizontal com imagem e título do evento.

### Próximos eventos
Essa seção mostra uma lista de eventos que irão acontecer em breve. Cada evento é apresentado em formato de card contendo:

- Imagem representativa do evento
- Nome do evento
- Data
- Localização
- Botão para visualizar mais detalhes

### Agenda
A área "Coloque na sua agenda" apresenta eventos organizados com foco na data e horário. O objetivo dessa seção é ajudar o usuário a visualizar rapidamente o que acontecerá em determinada data.

Cada card contém:

- Data
- Horário
- Nome do evento
- Botão para receber notificação ou lembrete

### Rodapé
O rodapé reúne links adicionais para navegação dentro do site, incluindo:

- Seções de eventos (esta semana, este fim de semana, este mês, este trimestre)
- Lista de cidades
- Categorias disponíveis
- Informações institucionais como a seção "Sobre nós"

## Objetivo do projeto

Este projeto foi criado com fins educacionais e tem como objetivo:

- Praticar construção de layouts modernos
- Trabalhar organização visual de conteúdo
- Desenvolver habilidades de estruturação de páginas web
- Aplicar conceitos aprendidos em cursos da Alura

## Observações

Este projeto foi desenvolvido como parte de um processo de aprendizado em desenvolvimento web e não representa um produto final de produção.
